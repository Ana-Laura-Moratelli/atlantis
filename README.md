<h2>:hammer_and_wrench: Como executar</h2>

  <h3>Passo 2: Instale as dependências</h3>

  <pre><code>npx tsc</code></pre>

  <h3>Passo 3: Execute o projeto</h3>
  <pre><code>cd src/js/app</code></pre>
  <pre><code>node app.js</code></pre>
